# Foodie

## Installazione:
  - Inserire il file .env nella cartella root del progetto.
  - Eseguire ```npm install```

## Uso:
Per lanciare correttamente l'applicazione bisogna eseguire i seguenti comandi su due terminali differenti:
      
-      npm run dev

-      npm run server
